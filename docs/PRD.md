# PRD - BabyAI (MVP)

## 1. 제품 개요
- 제품명(가칭): BabyAI
- 목표:
  - 음성 기반 육아 기록
  - 기록 기반 대화형 Q&A
  - 일일/주간 AI 리포트
  - 가족 사진 안전 공유(구독 기반)

## 2. 핵심 가치
- 즉시성: 기록 데이터 기반 즉답
- 자동화: 음성 입력 -> 이벤트 구조화 -> 차트/리포트
- 비용 통제: 사진 공유는 구독으로 제한

## 3. 사용자/권한 모델
### 3.1 Household 단위
- 아기, 구성원, 권한, 구독을 Household로 묶어 관리

### 3.2 역할
- `Owner`: 결제/권한/가족관리/전체 접근
- `Parent`: 기록/AI/사진(구독 범위)/커뮤니티
- `Family Viewer`: 사진 보기(다운로드는 옵션)
- `Caregiver`: 기록 입력(범위 제한), 사진 일부 접근(옵션)

### 3.3 권한 원칙
- 기본: 가족은 사진만 접근
- 기록/AI: 부모만 접근
- 다운로드: 기본 OFF, Owner가 제어
- 아기별 사진 접근 허용 지원

## 4. 가입/온보딩
### 4.1 부모 신규 가입(필수)
- Apple/Google/휴대폰 로그인
- 아기 프로필 최소 1개 생성(출생일 필수, 육아중 확인)
- 필수 동의: 약관/개인정보/데이터 저장·분석

### 4.2 가족 초대 가입
- 초대 링크/QR + 간편 로그인 + 최소 동의
- 아기 프로필 생성 강제 없음

### 4.3 실제 자녀 인증 정책
- MVP 미강제
- 추후 커뮤니티 신뢰 배지 등 선택 인증 도입

## 5. 페르소나/AI 톤
### 5.1 페르소나
- 부모 성향, 목표, 경고 민감도, 답변 형식, 커뮤니티 결합 ON/OFF

### 5.2 톤
- 친근/중립/격식/매우 간단히/코치형
- 적용 채널: 인앱 챗봇, 리포트, Siri/Bixby 응답

## 6. 핵심 기능
### 6.1 기록(필수)
- 이벤트 스키마:
  - 분유: `amount_ml`, `time`
  - 모유: `side`, `duration_min`, `time`
  - 수면: `start`, `end`, `duration_min`
  - 소변/대변: `count`, `stool_type`, `time`
  - 성장: `weight_kg`, `height_cm`, `time`
- 음성 UX: 녹음 -> STT -> 파싱 -> 카드 검수 -> 1탭 확정
- confidence 낮은 필드 강조 + 빠른 수정

### 6.2 차트/분석(필수)
- 기간: 오늘/7일/30일/사용자 지정
- 지표: 수유 총량·횟수·텀, 수면 총량, 배변 횟수, 성장 추세
- 미기록은 0이 아니라 미기록으로 처리

### 6.3 대화형 조회(필수)
- 지원: 팩트조회/계산/요약/비교
- 예:
  - 마지막 대변 시간
  - 다음 수유 ETA(최근 N회 평균 텀)
  - 오늘 요약
- 응답 규칙:
  - 근거 기준 명시(최근 N회 평균, 기록 범위)
  - 데이터 부족 시 불확실성 명시

### 6.4 음성 어시스턴트 연동(필수)
- iOS: App Intents
  - `GetLastPooTime`
  - `GetNextFeedingEta`
  - `GetTodaySummary`
  - `StartRecordFlow`
- 삼성: Bixby Result 대화 + 원격 endpoint + 커스텀 다이얼로그

### 6.5 AI 리포트(필수)
- 일일: 3줄 요약, 핵심 수치, 특이점, 내일 제안
- 주간(AI 구독): 평균/최대/최소/추세/행동 제안
- 의료 가드레일:
  - 진단/처방 금지
  - 응급 신호는 상담 권고
  - 기록기반 vs 일반정보 라벨 분리

### 6.6 가족 사진 공유(구독)
- 초대/앨범/업로드/보기/리액션(옵션)
- 다운로드 권한 토글
- 원본/고화질/썸네일 분리 저장 + CDN
- 저장공간/트래픽/초대 인원 플랜별 제한

## 7. 구독 플랜
- `Photo Share`: 사진 공유만
- `AI Only`: 챗봇 + 리포트 + 고급 분석
- `AI + Photo`: 통합 플랜
- `Free`: 기록/차트/커뮤니티 읽기 중심

## 8. MVP 포함/제외
### 포함
- 부모 가입 + 아기 프로필 강제 + 필수 동의
- 가족 초대 간소화
- 음성/텍스트 기록 + 파싱 + 확정 + 차트(오늘/7일)
- 인앱 AI 조회 3종(마지막 대변/다음 수유 ETA/오늘 요약)
- Siri/Bixby 조회형 인텐트
- 3종 구독 기능 게이팅

### 제외(2차)
- 포토굿즈 주문/물류 API
- 위치기반 의료 광고
- 중고마켓/에스크로

## 9. 비기능 요구사항
- 인앱 질의 체감 2~3초
- 기록/사진 접근 분리
- 감사 로그 필수(초대/권한/다운로드/구독변경)
