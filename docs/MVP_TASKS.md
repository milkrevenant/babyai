# MVP Tasks - BabyAI

## 0. 프로젝트 부트스트랩
- [ ] Flutter 앱 기본 구조 생성
- [ ] 백엔드(API) 프로젝트 생성
- [ ] PostgreSQL/Redis 로컬 개발 환경 구성
- [ ] CI 기본 파이프라인(린트/테스트) 구성

## 1. 인증/온보딩
- [ ] Apple/Google/휴대폰 로그인
- [ ] 부모 가입 플로우 + 아기 프로필 생성 강제
- [ ] 필수/선택 동의 저장
- [ ] 가족 초대 링크/QR + 토큰 검증 가입

완료 기준
- 부모 가입 시 아기 없이 완료 불가
- 가족 초대 가입은 간소화 경로로 완료 가능

## 2. Household/권한
- [ ] HouseholdMember role 모델링
- [ ] API 권한 미들웨어
- [ ] 아기별 접근 제어
- [ ] 다운로드 기본 OFF 정책

완료 기준
- 역할별 접근 제어 테스트 통과

## 3. 이벤트 기록(핵심)
- [ ] 이벤트 스키마 구현
- [ ] 음성 업로드 -> STT -> 파싱 API
- [ ] 확정 전 카드/신뢰도 반환
- [ ] 1탭 확정 저장 API
- [ ] 텍스트/빠른버튼 fallback

완료 기준
- 음성 1건에서 복수 이벤트 파싱/확정 가능

## 4. 차트/분석
- [ ] today/7d 집계 API
- [ ] 수유/수면/배변 핵심 지표 계산
- [ ] 미기록 구간 처리 로직
- [ ] Flutter 차트 화면 구현

완료 기준
- 미기록이 0으로 표현되지 않음

## 5. AI 조회/리포트
- [ ] 질의 분류기(조회/계산/요약/비교)
- [ ] 빠른 질의 3종 API
- [ ] 근거 기준 문장 자동 첨부
- [ ] 페르소나/톤 설정 저장 및 적용
- [ ] 일일/주간 리포트 생성
- [ ] 의료 가드레일 필터 적용

완료 기준
- 다음 수유 ETA 응답에 평균 텀 기준 문구 포함
- 데이터 부족 시 불확실성 안내 포함

## 6. Siri/Bixby
- [ ] iOS App Intents 4종 구현
- [ ] Bixby endpoint 연동 + result dialog 템플릿
- [ ] 톤 프로필 연계

완료 기준
- "마지막 대변", "다음 수유 ETA", "오늘 요약" 음성 질의 동작

## 7. 사진 공유/구독
- [ ] 앨범/업로드/보기 API
- [ ] 이미지 variant 생성(썸네일/중간/원본)
- [ ] 다운로드 토글
- [ ] 플랜 3종 게이팅 미들웨어

완료 기준
- 플랜별 기능 접근 차단/허용이 정확히 적용

## 8. 감사로그/NFR
- [ ] 초대/권한변경/다운로드/구독변경 로그 저장
- [ ] 핵심 쿼리 인덱스 점검
- [ ] AI 질의 캐시 적용
- [ ] 에러 모니터링 연결

완료 기준
- 핵심 사용자 행동이 audit 로그로 조회 가능

## 제안 일정(8주)
1. 1-2주차: 인증/온보딩/권한/데이터모델
2. 3-4주차: 기록(STT/파싱/확정) + 차트
3. 5-6주차: AI 조회/리포트 + Siri/Bixby
4. 7주차: 사진공유/구독 게이팅
5. 8주차: 안정화/성능/보안/스토어 배포 준비
